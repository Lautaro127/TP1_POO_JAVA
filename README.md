Repositorio del primer trabajo práctico de la materia de Java: una pequeña aplicación orientada a objetos que modela personas y empleados usando herencia, encapsulamiento y manejo de fechas con java.time. El proyecto contiene tres clases en el paquete dominio: Persona, Empleado y Principal; Persona define atributos básicos (DNI, nombre, apellido, fecha de nacimiento, género, dirección, teléfono y email) con constructores, getters, setters y un toString; Empleado extiende Persona añadiendo un número de legajo autoincremental y el puesto, además de un método estático para conocer el próximo legajo; Principal crea instancias de Empleado, las imprime por consola y muestra el siguiente legajo disponible.
